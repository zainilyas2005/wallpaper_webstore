Wallpaper Store Project
=======================

How to use:
1. Get a free Pixabay API key from: https://pixabay.com/api/docs/
2. Open assets/scripts.js and replace "YOUR_API_KEY" with your actual key.
3. Run the project:
   - Simple way: Just open index.html in your browser.
   - Better way: Serve with local server:
     * Python: python -m http.server 8000
     * Node: npx http-server
4. Search wallpapers and click Download to save them.
